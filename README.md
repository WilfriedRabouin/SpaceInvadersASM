# SpaceInvadersASM

<p align="center"><img src="screenshot.PNG"/></p>

## Prerequisites

- NASM
- GCC
- SDL2 v2.26.5
- SDL2_image v2.6.3
- SDL2_mixer v2.6.3

## Building

- Copy SDL2.lib, SDL2_image.lib and SDL2_mixer.lib in ext folder
- `make`

## Running

- Copy SDL2.dll, SDL2_image.dll and SDL2_mixer.dll in out folder
- `cd out`
- `./space_invaders.exe`
